What is the document object mode (DOM)?

The DOM (Document Object Model) is a programming interface for web documents, representing the page structure as a hierarchical tree of objects. It allows JavaScript to interact with and manipulate HTML or XML content dynamically.

How do you select an element from DOM?

To select an element from the DOM, you can use methods like `document.getElementById('id')`, `document.querySelector('.class')`, or `document.querySelectorAll('tag')`. These methods return the element(s) based on the selector you provide.

What is event delegation in the context of the DOM, and why it is useful?

Event delegation is a technique where a single event listener is attached to a parent element to handle events for its child elements. It improves performance and simplifies event handling, especially for dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using `element.setAttribute('attribute', 'value')` and retrieve them with `element.getAttribute('attribute')`. To change styles, use `element.style.property = 'value'` to set individual CSS properties directly.
